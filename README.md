# Customer Behavior Analysis  
**End-to-End Data Analytics Project | Python, SQL, Power BI**

## Overview
This project analyzes customer transaction data to uncover purchasing patterns, revenue drivers, and actionable insights for marketing, pricing, and customer retention. It reflects real-world data analyst work from raw data to business-facing dashboards.

---

## Business Objective
- Identify customer behavior patterns and revenue drivers  
- Improve data quality through systematic cleaning  
- Communicate insights via interactive dashboards  

---

## Dataset
- **Type:** Transactional customer data  
- **Records:** ~3,900 transactions  
- **Tools:** Python (Pandas), PostgreSQL  

---

## Approach
- Cleaned and validated raw data using Python  
- Performed exploratory data analysis to identify trends and anomalies  
- Used SQL (PostgreSQL) to calculate KPIs and analyze customer segments  
- Built interactive Power BI dashboards for stakeholder decision-making  

---

## Key Insight
> **A small group of repeat customers generates a disproportionate share of revenue and responds differently to discounts, indicating targeted promotions are more effective than blanket discount strategies.**

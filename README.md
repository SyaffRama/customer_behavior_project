# Data Analyst Portfolio Project | Python, SQL, Power BI
**Project Summary**

This project showcases an end-to-end data analytics workflow using Python, SQL (PostgreSQL), and Power BI. It demonstrates practical skills in data cleaning, exploratory data analysis (EDA), SQL querying, dashboard development, and business reporting.

The project simulates real-world data analyst tasks and highlights the ability to transform raw data into actionable insights for business decision-making.


# Business Objective
- Analyze structured business data to identify trends and performance drivers
- Improve data quality through systematic cleaning
- Deliver insights through interactive dashboards and professional reporting


# Dataset
- Type: Structured business dataset
- Format: CSV
- Usage: Data analysis, SQL querying, and visualization
- Processed Using: Python and PostgreSQL


# Skills Demonstrated
- Data Analysis
- Exploratory Data Analysis (EDA)
- Data Cleaning and Data Validation
- SQL (PostgreSQL)
- Data Visualization
- Dashboard Development
- Business Intelligence (BI)
- Data Storytelling and Reporting

# Tools and Technologies
- Python (pandas, numpy, matplotlib, seaborn)
- SQL / PostgreSQL
- Power BI
- Jupyter Notebook
- Gamma (Presentation and Reporting)

# Analytical Workflow
1. Data Loading and Preparation
  - Imported raw data using pandas
  - Validated schema, data types, and basic structure

2. Exploratory Data Analysis (EDA)
  - Analyzed distributions, trends, and correlations
  - Identified missing values, duplicates, and outliers
  - Generated summary statistics and visual insights

3. Data Cleaning
  - Handled missing and duplicate records
  - Standardized column naming and formats
  - Prepared analysis-ready datasets

4. SQL Analysis (PostgreSQL)
  - Loaded cleaned data into PostgreSQL
  - Wrote SQL queries for:
  - Aggregation and filtering
  - KPI calculation
  - Performance analysis
  - Optimized queries for dashboard use

5. Dashboard Development (Power BI)
  - Connected Power BI to PostgreSQL
  - Built interactive dashboards with KPIs and filters
  - Designed visuals for executive-level consumption

6. Reporting and Presentation
  - Summarized insights in a structured analytical report
  - Created a professional presentation using Gamma
  - Focused on clarity, insights, and business impact

# Dashboard Overview
The Power BI dashboard includes:
- Key Performance Indicators (KPIs)
